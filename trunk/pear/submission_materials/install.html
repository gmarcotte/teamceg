Install

The Pairgramming environment runs as a web service, and thus requires no installation on the part of the user. For web administrators seeking to set up their own Pairgramming server, some assembly is required.

Required Software

  • Apache Web Server
  • MySQL relational database
  • Python 2.4+ interpreter
  • MySQL-Python Bindings

With these basic tools in place, all necessary packages can be obtained from the Subversion repository hosted at: Pairgramming on Google Code In addition to our own software, this repository contains the following external packages:

  • Django, a Python web framework
  • Ajaxterm, a Javascript/Python terminal emulator
  • EditArea, a Javascript WYSIWYG editor designed for coding
  • Paramiko, a Python implementation of the SSH2 protocol
  • PyCrypto, a Python cryptography library
  • Pyjamas, a Python port of Google Web Toolkit
  • Pimentech, a 3rd party library for Django that provides nice JSON interaction

    • Check out a full copy of the trunk development branch onto your web server, and configure Apache to use Django as described: here

    Set up a MySQL user and database for use by Pairgramming, as described in the MySQL documentation.

    Then copy the pear/environments/prod/settings_prod.py file to pear/settings_local.py, and update all relevant fields to reflect your setup (e.g. hostnames, email servers, database, etc). Refer to Django documentation for more information about the settings configuration.

    Now run the vendor/install.py script to set up the 3rd party packages, and then run pear/environments/sandbox/initialize.py to set up the database and create a default user. This user's login info will be: email: pairgramming@gmail.com password: pw unless you modify it by changing the login script (highly recommended). You should also change the password manually after this is created.

    Now add the following lines to the virtual host Apache configuration file that you set up for Django:

    ProxyPass /ajaxterm/ http://localhost:8022/ ProxyPassReverse /ajaxterm/ http://localhost:8022/

    And start up the Ajaxterm daemon by navigating to pear/pjstuff/public/ajaxterm and running (as the www user):
    python ajaxterm.py -d

    You should now be off and running with your own Pairgramming system.
