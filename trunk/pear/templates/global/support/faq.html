The Pairgramming Application

Q. Can both people edit text in the editor or use the console at the same time? Doesn't that get confusing?
A. No, it does not get confusing because we don't allow both people to edit the editor and console at the same time. We followed the paired programming model truly, so that one person (who we call the driver) is editing and typing out the code, and the other person (who we call the passenger) is navigating and helping the driver design.

Q. How is this different from Google Docs?
A. Pairgramming provides much faster synchronization than Google Docs, so you see changes your partner is making in real-time. While both users can edit the file in Google Docs, our console and text editor allow only the driver to make changes, while the passenger watches. Finally, we have a full functionality console which you can use to work on any SSH-able development server. Also, our logo is cuter.
